Info & Settings

GitHub Gist — credentials

Your recipes are stored in a GitHub Gist as a recipes.json file. Enter your Gist ID and personal access token below to connect.

Stored only in your browser's localStorage. Never sent anywhere except GitHub's API directly.

Setting up GitHub Gist

  1. Create a free account at github.com
  2. Go to gist.github.com — create a new secret Gist
  3. Name the file recipes.json, put [] as content, save
  4. Copy the Gist ID from the URL (the long string at the end)
  5. Go to github.com/settings/tokens → Generate a classic token with the gist scope only
  6. Paste both above and hit Save

Using on iPhone

  • Save this HTML file to your iCloud Drive
  • Open it from the Files app in Safari
  • Tap Share → Add to Home Screen to install it like an app
  • Recipes load from GitHub and cache locally — works offline with the last synced data

Offline mode

When offline, you can still browse and edit recipes. Changes are queued and synced to GitHub automatically when your connection returns. A banner at the top shows your current status.
